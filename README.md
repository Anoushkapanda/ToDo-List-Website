# ToDo-List-Website

A clean and responsive to-do web app built using **HTML**, **CSS**, and **TailwindCSS + Flowbite** for reusable UI components.  
This is a learning and portfolio project focused on building scalable, maintainable, and modular frontend code.

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Flowbite Components
- Vanilla JavaScript
- Custom CSS
